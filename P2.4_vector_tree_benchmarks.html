2.4.- Benchmarks  countertree::vector_tree vs  std::deque

In order to evaluate the performance, we compare the countertree::vector_tree with the STL class std::deque, which is a vector with fast insert at the beginning and the end.

  • In the first test the elements are integer, and there are two test : insert/delete in the first or the last position and insert/delete in random positions.The test is done with structures with 100 elements anwith 1000000 elements
  • In the second test the elements are a set of integers with 100 elements, and there are two test : insert/delete in the first or the last position and insert/delete in random positions.The test is done with structures with 100 elements anwith 10000 elements

 

The test had been done with  GCC 4.6 (64 bits) and run on a QuadCore 2.4 GHz with the C++0x actived (without this option the deque times of the second test are 40 times bigger).
